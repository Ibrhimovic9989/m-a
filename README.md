# Muneer & Associates

Editorial-luxury marketing site for Muneer & Associates — a Hyderabad-based chartered accountancy practice (est. 2012).

Built with Next.js 14 (App Router), TypeScript, Tailwind CSS, and Geist Sans.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Production build

```bash
npm run build
npm run start
```

## Deployment

Configured for Vercel via `vercel.json`. Push to `main` and import the repo at https://vercel.com/new — no environment variables required.
